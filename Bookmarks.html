Bookmarks:

The following tools (bookmarklets) allow you to conduct a range of searches / tasks, directly from your web browser, without needing to log into the OSINT Toolkit. To use any of the bookmarklets, below, simply drag the link(s) into your 'Bookmarks Toolbar'.

Note, some of the bookmarklets only work on certain websites or during certain tasks so please read the description(s) carefully before dragging the link(s) into your 'Bookmarks Toolbar'. If you want to access and use all of the bookmarklets below, except for the Youtube Downloader and View Selection, just drag the Toolkit B.D. bookmarklet into your ‘Bookmarks Toolbar’.

Bookmark Description
Toolkit B.D. This bookmarklet allows you to access and use all of the bookmarklets below, except for the Youtube Downloader and View Selection, in a single options menu from your web browser without the need to install each bookmark individually.
All Images This bookmarklet extracts all images that contain external links on the current page and displays them in a new window.
Cluster Search This bookmarklet will conduct a cluster search on the word / phrase specified and will group returned data into thematic categories. N.B. replaces current window when used.
Display All Links This bookmarklet will extract all external links on the current webpage and display them in a new window.
Facebook (FB) Search This bookmarklet will replace the current window and search Facebook (FB) for the word or phrase specified. N.B. Being logged into a FB account prior to use will achieve the best results.
FB ID This bookmarklet will display the FB ID number of the user’s profile you are currently viewing. N.B. You must refresh the page (F5) prior to use.
FB Comment Expander This bookmarklet will expand every comments section, on the user’s profile you are currently viewing, to show hidden comments and, where applicable, will automatically translate all comments to English.
Find on this Page This bookmarklet will search the current page for the word or phrase specified. If found, every occurrence will be highlighted yellow and a outline of how many times the word or phrase was found will be displayed.
Int / Ext Links This bookmarklet will colour links on the current page one of three colours depending on their attributes; Red for internal links, Blue for external links and Orange for in-page links.
Last Modified This bookmarklet finds the last modified date of the current webpage. N.B. if a webpage is dynamic, this bookmarklet will show the current date and time.
List Email Links This bookmarklet will outline any email addresses prefixed with a 'mailto tag' on the current webpage and display this information in a new window.
Search Google Cache This bookmarklet will search Google Cache for the website specified. N.B. replaces current window when used.
Search Internet Archive This bookmarklet will search the Internet Archive for the website specified. N.B. replaces current window when used.
Search Links This bookmarklet will search all the links on the current page for the word or phrase specified and will list, if applicable, any results in a new window. N.B. This bookmarklet searches both the link text and target URL for any matches.
Search Popular Social Media This bookmarklet will search a range of popular social media sites, via a CSE, for the word or phrase specified. N.B. replaces current window when used.
Search Twitter This bookmarklet will search Twitter for the word or phrase specified. N.B. replaces current window when used.
Twitter Analytics (keyword) This bookmarklet will provide analytical data (followers, location data, user information, etc) on all tweets featuring the word or phrase specified. N.B. replaces current window when used.
Twitter Analytics (username) This bookmarklet will provide analytical data (followers, location data, user information, etc) for the user account specified. N.B. replaces current window when used.
Translate to English This bookmark will translate the current webpage to English. N.B. replaces current window when used.
View Selection This bookmarklet copies any selected text into a new window. N.B. It preserves links, paragraphs and images.
Website Information Search This bookmarklet displays a range of information about the website specified, including IP addresses / Subdomains used by the website. N.B. replaces current window when used.
Reverse Whois Search This bookmarklet checks for information relating to the name, address, telephone number, email, etc of the website specified. N.B. replaces current window when used.
Word Frequency This bookmarklet will count all the text on the current webpage and will display, in a new window, how many times each word features.
Youtube Video Downloader This bookmarklet will download any Youtube video into a storyboard format. N.B. You should select the Youtube video you want to download prior to using this bookmarklet. There is no requirement to watch the video you want to download.
